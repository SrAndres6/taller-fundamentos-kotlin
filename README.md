# Taller de Fundamentos de Kotlin

## Nombre completo: Juan Andres Jaramillo Garcia

## Descripción

Este repositorio contiene el desarrollo de **30 ejercicios prácticos en Kotlin**, realizados con el objetivo de fortalecer los conocimientos fundamentales del lenguaje y comprender su sintaxis moderna.

Los ejercicios fueron organizados por bloques temáticos, permitiendo trabajar progresivamente desde conceptos básicos hasta el manejo de colecciones y funciones de extensión.

## Objetivo

El propósito principal de este taller fue:

- Comprender la sintaxis básica de Kotlin.
- Practicar el uso de variables y tipos de datos.
- Implementar seguridad de nulos (*Null Safety*).
- Aplicar estructuras condicionales.
- Utilizar ciclos y repeticiones.
- Crear funciones con parámetros.
- Manipular colecciones.
- Desarrollar extensiones personalizadas.
